# neworbit-tslint-config

TSLint configuration for NewOrbit

## Installation

`npm install neworbit-tslint-config --save-dev`

## Setup

Add `tslint.json` file with the following contents:

```json
{
    "extends": "neworbit-tslint-config"
}
```
